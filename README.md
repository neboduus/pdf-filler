# pdf-filler

A simple module written in Python to fill your PDF using PDF Forms and fillpdf==0.7.2

## Requirements

- Python 3.8.2

## Create venv and install dependencies

```shell
make init
```

## How to use

- See `test_fill` in `tests/test_pdf_filler.py` for a full working example

### How to test

```shell
make test
```
